Amazon Lightsail is a virtual private server (VPS) provider and is the easiest way to get started with AWS for developers, small businesses, students, and other users who need a solution to build and host their applications on cloud. - https://aws.amazon.com/lightsail/faq/

Access command-line interface

The user needs to have full access or specific access to Lightsail actions.

  1. Create a policy, search for service lightsail
  2. Allowed actions
  1. Tag if you want (it is best practice)
  2. Create policy as CarlLightsailFullAccess
  3. Attach the new policy to your User Group under permissions
  • Actions, resources, and condition keys for Amazon Lightsail
