Futaba 10j

Aux-Chan

To assign a switch to a channel, here I use CH6 / SwC for flaps.

Menu 1 – AUX-CHAN – CH6 – SwC

  • Manual page 56

End Point

To set servo end points if they are mechanically too far for the current TX/RX.

Menu 1 – END POINT – 6:FLP (on pg2) – <- =72

  • Manual page 112

Dual Rate / EXPO

Menu 3 – D/R EXPO

These are conservative settings that worked well with my apprentice and H9-40 stick.

| 1 | EXP | -30% | -40% |

  • Manual page 104

Mixing flaps to work with aileron

My H9-40 sticks flaps travel was the same as the ailerons so this would effectively create larger ailerons.

  1. Set all AUX-CHAN 5to10 to NULL, this released/centered the servos

  2. Enable FLAPARONS (Menu 3), this mixed 1 & 7 in opposite directions (Could also be done with mixing)

  3. Enable under menu 1 P.MIX 1-6 two mixes
