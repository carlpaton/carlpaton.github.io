Mixing flaps to work with aileron

My H9-40 sticks flaps travel was the same as the ailerons so this would effectively create larger ailerons.

  1. Set all AUX-CHAN 5to10 to NULL, this released/centered the servos

  2. Enable FLAPARONS (Menu 3), this mixed 1 & 7 in opposite directions (Could also be done with mixing)

  3. Enable under menu 1 P.MIX 1-6 two mixes

Redis

Redis is an open source (BSD licensed), in-memory data structure store, used as a database, cache and message broker. You can quickly spin up a container and have a hoon:

  1. Create a network for the containers to connect to.
  1. Bring up the Redis Server
  1. Bring up redis administrator which is my open source UI for Redis

This UI can then be seen at - http://localhost:8080/

  • https://github.com/carlpaton/RedisAdministrator
  • https://hub.docker.com/r/carlpaton/redis-administrator
  1. Bring up redis commander which is an open source UI for Redis written by Joe Ferner.

This UI can then be seen at - http://localhost:8081/
