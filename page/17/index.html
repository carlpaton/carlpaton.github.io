Dual-Boot Windows 10 & Ubuntu 18.04 - 16.04

The steps are summarized as follows:

  1. Install windows 10 first
    1. Use the maximum disk space for the volume as NTFS
    2. Windows USB/DVD Tool
  2. Install Ubuntu second
    1. At the disk/volume option choose ‘alongside’ windows 10
    2. Create a new ‘ext4’ partition using the wizard.
    3. UNetbootin
