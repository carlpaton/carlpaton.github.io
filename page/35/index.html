Senior Telemaster

Based on suggestions by domain experts the wing should be built with zero dihedral: https://en.wikipedia.org/wiki/Dihedral_(aeronautics)

Wing Construction (2 piece, removable spar):

  • Wing ribs, stab ribs & tips are 7~OF 3.0mm x 100mm x 915mm ~ 1/8”
  • Spars are spruce/basswood 4~OF 9.5mm x 12.5mm x 915mm (3/8” x 1/2”)
  • Center ribs where wings join 1~OF Poplar Plywood 3.0mm x 300mm x 300mm (1/8”)

Wing Construction (1 piece):

  • Spar is balsa 1~OF 9.5mm x 12.7mm x 121.92cm (3/8” x 1/2” x 48”) cut into strips 9.5mm x 19 mm (3/8” x 3/4”)

C# Enum

Enumeration types

The enum keyword is used to declare an enumeration, a distinct type that consists of a set of named constants called the enumerator list.

  • https://docs.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/enum

Enumeration types as bit flags

  • https://github.com/carlpaton/EnumDemo

You can use an enumeration type to define bit flags

You create a bit flags enum by applying the System.FlagsAttribute attribute and defining the values appropriately so that AND, OR, NOT and XOR bitwise operations can be performed on them.

  • https://docs.microsoft.com/en-us/dotnet/csharp/programming-guide/enumeration-types#enumeration-types-as-bit-flags
