Raspberry Pi Cluster - Docker Swarm

WIP!

Docker Swarm is not dead, its simple to setup and has a shallow learning curve.

Following on from the post Raspberry Pi Cluster which created a cluster of PI’s and used parallel computing to execute and share the workload of some Python scripts I figured it was time to do the same thing using Docker Swarm.

“Docker Swarm is an alternative, Docker-native Container Orchestration Engine that coordinates container placement and management among multiple Docker Engine hosts. Docker Swarm allows you to communicate directly with swarm instead of communicating with each Docker Engine individually. Docker Swarm architecturally is comprised of two types of nodes called Managers and Workers. -rancher.com”

Hardware Setup

  1. Download Raspberry Pi OS (32-bit) Lite - 2020-08-20-raspios-buster-armhf-lite
  2. Install on SD cards with Etcher
  3. Boot Pi, allow disk expand, config country, Wifi and SSH, rename the Pi’s (This can be done with sudo raspi-config)

I called them:

  • master-01
  • node-01
  1. Install additional software
  1. Setup SSH keys and test primenumbers27.py, probably didnt need to do that step but its cool to see it work again :D

Docker & Swarm Setup

Install Docker on all nodes using the PI user.
