Chrome Extensions

From Chrome, run chrome://extensions/ or https://chrome.google.com/webstore/category/extensions

  • Also see Netify
    • https://chrome.google.com/webstore/detail/netify/mdafhjaillpdogjdigdkmnoddeoegblj
  • Ignore X-Frame headers
    • Also see X-Frame-Options
  • Allow-Control-Allow-Origin: *
    • Allows to you request any site with ajax from any source. Adds to response ‘Allow-Control-Allow-Origin: *’ header
  • UnXSS
    • Intercept and modify or delete websites’ security headers
  • Postman Interceptor
    • Sends requests fired through the Postman chrome app.
  • Redux DevTools
    • Query store
  • Fireshot
    • Screenshot web pages
  • Wappalyzer
    • Provides information about the page you visited, frameworks, hosting ect
    • https://www.wappalyzer.com
  • Edit this cookie
    • Edit cookies, nom :D
    • https://chrome.google.com/webstore/detail/editthiscookie/fngmhnnpilhplaeedifhccceomclgfbg?hl=en
  • Mozbar
    • Display SEO related elements such as meta tags
    • https://chrome.google.com/webstore/detail/mozbar/eakacpaijcpapndcfffdgphdiccmpknp?hl=en
