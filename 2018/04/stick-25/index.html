Big Stick Laser Cut Balsa Kit 1060mm

The HK link says its 1060mm but the wing I got is 1200 without the tips so I’ll go with the slightly longer size. These are some guestimates on wood to be used.

Fuselage

  • Ply re-enforcement where the landing gear attaches.
  • F1 -> 5 or 6mm ply
  • F2,3 -> 3mm ply
  • F4,5,6 -> 5mm balsa
  • Triangle stock -> Probably cut from 3mm balsa scraps
  • Gussets -> 2 or 3mm square
  • Sides -> 2mm balsa forward grain
  • Top/Bottom -> 2mm balsa sideways grain
  • Wing seats 2mm balsa vertical grain (Thanks for the tip bro Clive!)

Wing

  • Main spar & LE -> 8mm square hard balsa
  • Forward spar/stringers -> 4mm balsa
  • Sheer web -> 2mm balsa
  • Ribs & caps, 2mm balsa
  • Wing tips, 5 or 6mm balsa
  • Sheeting -> 2mm balsa
  • TE -> 6mm square (I think)
  • Triangle stock -> Probably cut from 3mm balsa scraps

Tail

  • 4mm balsa, I’ll build it up with 2mm to try save weight.
